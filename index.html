Sailboat state equations

The goal of this class is to implement a sailboat simulation based on the state equations proposed in "Mobile Robotics, Luc Jaulin, 2015."

You are given a copy of this book and a ready to import simulation environment available at:Sailboat github repository

The environment contains

  1. a 3D model of L'improbable ENSTA Bretagne robot sailboat.
  2. a ready to fill sailboat actuator

Tasks

  1. Modify the actuator so that when the sail and rudder angle are changed, it is reflected on the 3D model [1]
  2. Withe help from [1],[2], code and integrate the state equations of the sailboat in the actuator.
  3. Test the actuator and submit a pull request to http://github.com/nicolaje/sailboat.
